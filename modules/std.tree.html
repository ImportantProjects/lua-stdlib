clone (t, nometa)
Make a deep copy of a tree, including any metatables.

To make fast shallow copies, use std.table.clone.

Parameters:

  • t table 
      table to be cloned
  • nometa boolean if non-nil don't copy metatables

Returns:

    table a deep copy of t
